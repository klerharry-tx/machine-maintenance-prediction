# 🔧 Machine Maintenance Prediction System

## 📌 Overview
This project predicts machine condition (Normal or Fault) using sensor data and machine learning. It helps in predictive maintenance to reduce machine breakdowns and improve efficiency.

---

## 📊 Dataset
- fault.csv → sensor data of faulty machines  
- normal.csv → sensor data of normal machines  

---

## ⚙️ Workflow
1. Data loading and preprocessing  
2. Model training using machine learning algorithm  
3. Saving trained model as `motor_model.pkl`  
4. Real-time prediction using saved model
